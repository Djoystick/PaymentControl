# Support

Thanks for using Payment Control.

## Where to Ask for Help

Use one of these channels:
1. GitHub Discussions (if enabled in this repository) for usage questions and Q&A.
2. GitHub Issues for reproducible product bugs or clear improvement requests.

If Discussions are not enabled, open an issue and prefix the title with `[Support]`.

## Where to Report Bugs

Please use the bug report template:
- `.github/ISSUE_TEMPLATE/bug_report.yml`

Include:
1. What happened.
2. Steps to reproduce.
3. Expected behavior.
4. Environment details (device/browser/Telegram client).

## Feature Requests

Please use the feature request template:
- `.github/ISSUE_TEMPLATE/feature_request.yml`

Keep requests practical and scoped to a clear user problem.

## Security Issues

Do **not** report security vulnerabilities in public issues.

Use the process in [SECURITY.md](./SECURITY.md).

## Scope Reminder

Support channels are for product/runtime questions, bug reports, and feature requests.
They are not for private credentials, secrets, or account-sensitive data.
